Enumerable Methods: Map

July 18, 2015

map {|obj| block} ==> array

The ruby docs exlain that the map method "returns a new array with the results of running block once for every element in enum

What does this mean? Say we have an array:

array = [2, 2, 2, 2]

Suppose we want to add 1 to every element of the array. We could use the map method to accomplish this:

array.map {|obj| obj + 1}

In other words, map allows us to iterate over each element of an array, and transform each element of the array by applying a specific operation to it as specified by block

One distinction to make is between "map" and "map!". "map" will not modify the original array and will instead return a new array with the modified values "map!" will modify the original array.
